standard readme for setup on installation for competition

Welcome to our project, Draconic Elements! Here is how you install the game:
1)	Unzip the files from the container zip file into a folder of your choice.
2)	Open the installation wizard called setup.
3)	Select your language you prefer.
4)	Accept the fake end user license agreement and hit next.
5)	Optionally, you can create a desktop shortcut.
6)	Click install.
7)	Click launch or select the .exe file.
